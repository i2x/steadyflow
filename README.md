# DCA Rotation Planner

เว็บแบบเรียบง่ายสำหรับ GitHub Pages

- ไม่มี input
- ดูวันปัจจุบันอัตโนมัติ
- คำนวณต่อเนื่องไม่จำกัดปี
- แสดงย้อนหลัง 2 สัปดาห์ และล่วงหน้า 2 สัปดาห์
- แสดงตัวนับถอยหลังไปหุ้นตัวถัดไป

ค่าเริ่มต้นที่ล็อกไว้:
- เริ่มต้น: 2026-03-30
- งบ: 10,000 บาท/เดือน
- ซื้อ 4 ครั้ง/เดือน
- งบต่อสัปดาห์: 2,500 บาท
- พอร์ตเป้าหมาย: O 35% / ENB 40% / BIP 25%
- สัปดาห์แรกเริ่มที่ O

## Deploy

1. สร้าง GitHub repository ใหม่
2. อัปโหลด `index.html` ไปที่ root ของ repo
3. ไปที่ Settings > Pages
4. เลือก Deploy from a branch
5. เลือก branch `main` และ folder `/root`
